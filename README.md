# QurbaniHat – Livestock Booking Platform

QurbaniHat is a premium, state-of-the-art livestock booking platform designed to streamline the process of purchasing sacrificial animals for Qurbani. By connecting buyers with verified farms, the platform ensures transparency, health standards, and a seamless digital booking experience.

## 🌐 Live URL
[https://qurbanihat.com](https://qurbanihat.com)

## 🎯 Purpose
The primary goal of QurbaniHat is to modernize the traditional livestock market. It provides a reliable digital marketplace where users can browse, select, and book healthy animals with confidence, avoiding the complexities of physical cattle markets while ensuring Shariah compliance and animal welfare.

## ✨ Key Features
- **Verified Livestock Listings**: Explore a curated selection of healthy cows, goats, and sheep with detailed specifications and farm information.
- **Secure Authentication**: Robust user authentication and session management powered by **Better-Auth**, supporting social logins.
- **User Dashboard & Profile**: Personalized experience where users can manage their information and track booking activities.
- **Premium UI/UX**: A modern, responsive interface featuring:
  - **Glassmorphism & Vibrant Design**: A high-end aesthetic with curated color palettes.
  - **Dark Mode Support**: Seamless transition between light and dark themes.
  - **Smooth Animations**: Interactive elements using **Framer Motion** and **Animate.css**.
- **Qurbani Resources**: Comprehensive guides on Shariah guidelines, animal selection tips, and post-delivery care.
- **Dynamic Feedback**: Real-time status updates with **React-Toastify** and professional loading states with **React-Spinners**.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.

## 🛠️ Technology Stack & Packages

### Core Framework
- **[Next.js 15+](https://nextjs.org/)**: The React framework for production.
- **[React 19](https://react.dev/)**: For building a dynamic and interactive UI.

### Authentication & Database
- **[Better-Auth](https://better-auth.com/)**: Comprehensive authentication library for Next.js.
- **[MongoDB](https://www.mongodb.com/)**: NoSQL database for flexible data storage.
- **[@better-auth/mongo-adapter](https://www.better-auth.com/docs/adapters/mongodb)**: Database adapter for authentication.

### Styling & UI Components
- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS framework.
- **[DaisyUI](https://daisyui.com/)**: Component library for Tailwind CSS.
- **[HeroUI](https://heroui.com/)**: High-performance UI components.
- **[Shadcn/UI](https://ui.shadcn.com/)**: Reusable components built with Radix UI.
- **[Lucide React](https://lucide.dev/)** & **[HugeIcons](https://hugeicons.com/)**: Beautifully crafted icon sets.

### Animations
- **[Framer Motion](https://www.framer.com/motion/)**: For advanced layout and gesture animations.
- **[Animate.css](https://animate.style/)**: For pre-built, cross-browser CSS animations.
- **[Tw-Animate-CSS](https://github.com/v-v-e/tw-animate-css)**: Tailwind integration for CSS animations.

### State Management & Forms
- **[React Hook Form](https://react-hook-form.com/)**: Performant, flexible, and extensible forms.

### Utilities & Feedback
- **[React-Toastify](https://fkhadra.github.io/react-toastify/)**: Elegant notifications for user feedback.
- **[React-Spinners](https://www.davidhu.io/react-spinners/)**: Loading animations for better UX.
- **[Clsx](https://github.com/lukeed/clsx)** & **[Tailwind-Merge](https://github.com/dcastil/tailwind-merge)**: For dynamic class management.

### Development Tools
- **[JSON-Server](https://github.com/typicode/json-server)**: Mock REST API for rapid development and testing.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
